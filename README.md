# Name
sakura-app

Basic認証を使用しているため、chromeかsafariでログインをお願いいたします。

# URL
https://sakura-app-ko6p.vercel.app/ <br >
トップページのログインボタンから、メールアドレスとパスワードを入力してログインできます。
